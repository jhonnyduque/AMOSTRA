# AMOSTRA

AMOSTRA es una plataforma de portafolios profesionales para estudiantes y creativos. Permite mostrar trabajos reales de formación, recibir retroalimentación estructurada y compartir un perfil público como evidencia de aprendizaje y habilidades.

## Cómo correr local

1. Clona o descarga este repositorio.
2. Abre `index.html` en tu navegador (doble clic o arrastrar al navegador).

## Qué incluye el MVP

- Encabezado de perfil con nombre, rol/área y frase profesional.
- Resumen corto visible del enfoque del portafolio.
- Lista de trabajos ordenados por fecha.
- Cada trabajo con descripción, contexto de aprendizaje integrado, etiquetas y 1–3 imágenes.
- Retroalimentación estructurada con fortalezas, áreas de mejora y siguientes pasos.
- Datos mock editables en `data.js` (puedes cambiar nombre, trabajos y feedback).
- Diseño limpio y responsive básico.
